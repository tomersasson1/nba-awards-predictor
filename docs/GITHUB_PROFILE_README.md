# Hi, I'm Tomer

I'm a Data Science student focused on Python, machine learning, and end-to-end data pipelines. I build portfolio projects that combine data engineering, ML modeling, and interactive dashboards.

## Featured Projects

### [NBA Awards Predictor](https://github.com/tomersasson1/nba-awards-predictor)
I built an end-to-end ML pipeline that predicts NBA award voting (MVP, DPOY, ROTY, MIP, 6MOY, COTY). It uses PyTorch, web scraping from Basketball Reference, and a Plotly Dash dashboard. I engineered ~47 features, implemented award-specific eligibility filters (e.g. true rookies for ROTY, bench players for 6MOY), and added optional Google Trends integration for media-hype signals.

### [CineMatch](https://github.com/tomersasson1/cinematch)
A movie and TV recommendation system I built using collaborative filtering, content-based models, and a Streamlit UI. It runs on MovieLens 25M and combines popularity-based baseline, item-item CF, and genre-based signals into a hybrid recommender.
