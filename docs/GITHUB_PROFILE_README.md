# Hi, I'm Tomer

Data Science student focused on Python, machine learning, and end-to-end data pipelines. I build portfolio projects that combine data engineering, ML modeling, and interactive dashboards.

## Featured Projects

### [NBA Awards Predictor](https://github.com/tomersasson1/nba-awards-predictor)
End-to-end ML pipeline predicting NBA award voting (MVP, DPOY, ROTY, MIP, 6MOY, COTY) using PyTorch, web scraping, and Plotly Dash. Features ~47 engineered stats, award-specific eligibility filters, and optional Google Trends media-hype integration.

### [CineMatch](https://github.com/tomersasson1/cinematch)
Movie & TV recommendation system with collaborative filtering, content-based models, and a Streamlit UI. Built on MovieLens 25M with a hybrid approach combining popularity, item-item CF, and genre-based signals.
